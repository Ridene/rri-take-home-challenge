# rri-take-home-challenge

You can run the application with 'npm' or 'yarn'

### Install dependencies
``` npm install ```
or
``` yarn install ```

### Run application on local
``` yarn dev ``` or 
``` npm run dev ```

### build application
``` yarn build ``` or 
``` npm run build ```

### Execute eslint
``` yarn eslint ``` or 
``` npm run eslint ```
